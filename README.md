# Мои DevOps-скрипты

Это моя коллекция скриптов для мониторинга и автоматизации серверов на Linux (WSL).

## 📁 Скрипты

- **`check_server.sh`** — проверяет статус Nginx
- **`health_check.sh`** — показывает состояние CPU, RAM и диска
- **`ultimate_monitor.sh`** — собирает всё вместе и отправляет отчёт в Telegram
